# _compare-and-patch_

Usage:

`compare-and-patch [options]`

`cap [options]`

## options:

`--help (-h)`

Outputs all possible flags.

`--origin (-o)`

**Required**

Path to the origin folder.

For example: `./origin`

`--target (-r)`

Path to the target folder.

For example: `./path/to/target`

`--silent (-s)`

Silence all logs and errors.

`--verbose (-v)`

Outputs all possible info.
